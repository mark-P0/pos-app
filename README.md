A point-of-sales and inventory management application

## Libraries | Frameworks

- `electron-vite`
  - Electron
  - Vite
  - React
  - Typescript

---

- Prettier
- ESLint

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For Windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
