# Point-of-Sales

The creation of a point-of-sales application is a very common task given to IT and CS students in my locale. In this project, I will attempt to create something similar using JavaScript.

## Frameworks & Libraries

These are the frameworks and libraries that I expect to be using. This list may grow in the future!

- **React**. General tech the app is based on.
- **Chakra UI**. React UI component library.
- **Firebase**. For cloud data storage and authentication.
- **Electron**. For bundling the app as a desktop executable.
